## 使用方法：                                 
1.将所有的基因型文件放在02文件夹中                   
  例如"TraesCS5A03G0123456.filter.vcf.gz
2.将表型文件放在05文件夹中，命名为trait.txt          
  第一列名称为ID，后面每一列代表一个表型，例如"HT32L"      
3.软件自动识别基因与表型信息                       
4.在当前文件夹下执行". ./start.sh"             
5.结果将在后续生成                            
6.初始化与清除工作空间请执行". ./clearn.sh"
